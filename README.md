# 🐦 Twitter Clone (Fullstack + Docker + GitHub Actions)

A simple fullstack Twitter Clone built with **React**, **Node.js**, **MongoDB**, and containerized with **Docker**. Includes a basic CI pipeline using **GitHub Actions** and is runnable locally for easy screenshots and portfolio demonstration.

---

## ⚙️ Tech Stack

- **Frontend**: React (Create React App)
- **Backend**: Node.js + Express
- **Database**: MongoDB (Docker container)
- **CI/CD**: GitHub Actions
- **Orchestration**: Docker Compose

## 🚀 Features

- Tweet submission with `user` and `content`
- Real-time feed of all tweets
- Fully containerized backend/frontend/db
- CI pipeline for dependency check and linting
